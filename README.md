# E-Commerce-Dimensional-Modelling-And-Data-Analysis
This project presents a comprehensive analysis and data-modeling workflow designed to understand and optimize the order management processes of an e-commerce business. Leveraging the [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce), the study evaluates operational performance across five core business functions and builds an efficient Power BI data model to support data-driven decision-making.
